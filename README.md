## 功能说明

- 输入待做事件后 回车加入待做列表中
- CheckBox 可以选中,表示已完成
- 每个事件右边也有删除按钮
- 底部信息条显示
  - 左边显示 剩余未完成事件
  - 中间选中 事件状态 
  - 右边清除所有的已完成事件

## 项目运行

安装依赖包:

`npm install`

运行项目:

`npm run dev`
